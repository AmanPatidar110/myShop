2. MongoDB Sessions
3. Login
4. AddToCart
5. Order
6. Invoice
7. Empty Cart
8. Add Product
9. Pagination
10. Edit Products
12. SignUp (Validation, password encry.)    
13. SendingEmail
14. Reset Password
